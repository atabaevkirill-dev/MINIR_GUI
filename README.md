# MINIR Термокамера - Контроллер

Приложение для управления тепловизионной камерой MINIR по протоколу RS-232.

## Особенности

- Подключение к термокамере через последовательный порт (RS-232)
- Управление всеми функциями камеры согласно протоколу
- Графический интерфейс с интуитивным расположением элементов управления
- Поддержка всех команд протокола: шторка, калибровка, яркость, усиление, фильтры и т.д.
- Возможность выбора скорости передачи данных (9600 или 19200 бод)
- Логирование отправленных команд
- Темный интерфейс в стиле Visual Studio Code
- **Кроссплатформенная поддержка**: Windows, macOS и Linux

## Требования

- Python 3.6 или выше
- Библиотека pyserial
- PyQt6 (6.4.0 или выше)

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Подключите термокамеру к компьютеру через RS-232 адаптер

3. Запустите приложение:
   ```
   python main_pyqt_claude.py
   ```
   
   Или используйте кроссплатформенный скрипт запуска:
   ```
   python launch_crossplatform.py
   ```

## Использование

1. Выберите правильный COM-порт из выпадающего списка
2. Выберите скорость передачи данных (по умолчанию 9600 бод)
3. Нажмите "Подключить"
4. Используйте вкладки для управления различными функциями камеры:
   - Базовое управление: шторка, калибровка, сброс
   - Управление изображением: яркость, усиление, DDE, фильтры, перекрестие
   - Расширенное управление: цветовые палитры, зум, режимы изображения, гамма
   - Ручное управление: переключатели, дополнительные функции

## Поддерживаемые функции

- Управление шторкой (открытие/закрытие)
- Калибровка (ручная, фоновая, автоматическая)
- Регулировка яркости (0-255)
- Регулировка усиления (0-255)
- DDE (Dynamic Differential Enhancement) (0-255)
- Пространственный фильтр (0-255)
- Управление перекрестием (X: 0-701, Y: 0-575)
- Цветовые палитры (0-11)
- Электронный зум (1x, 2x, 4x)
- Режимы изображения (L, M, H)
- Инверсия изображения (горизонтальная, вертикальная)
- Гамма-коррекция (0-23)
- NUC таблицы (0-2)
- Переключатели (перекрестие, фильтры, улучшение изображения, белый горячий, AGC)
- Дополнительные функции (автофокус, запрос статуса, two point calc и др.)

## Настройки последовательного порта

- Скорость передачи: 9600 бод (по умолчанию) или 19200 бод
- Биты данных: 8
- Стоповые биты: 1
- Проверка четности: нет

## Кроссплатформенная информация

Приложение работает на Windows, macOS и Linux:

- **Windows**: COM-порты (COM1, COM2 и т.д.)
- **macOS**: устройства типа /dev/cu.usbserial-*, /dev/tty.usbmodem*
- **Linux**: устройства типа /dev/ttyS*, /dev/ttyUSB*, /dev/ttyACM*

**Важно для Linux/macOS**: Может потребоваться добавить пользователя в группу dialout или использовать sudo для доступа к последовательным портам.

## Протокол связи

Приложение полностью соответствует протоколу связи, описанному в документации на термокамеру MINIR.
